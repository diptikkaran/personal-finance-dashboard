# Personal Finance Dashboard

A full-stack banking dashboard that allows customers and employees to manage accounts and transactions.

## Features

* Customer Registration
* Customer Login
* Employee Login
* Deposit Requests
* Withdraw Requests
* Transaction History
* Account Management
* Balance Tracking
* Admin Dashboard

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript
* React

### Backend

* Spring Boot
* Java

### Database

* SQL
